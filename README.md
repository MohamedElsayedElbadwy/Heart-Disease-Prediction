Heart Disease Prediction
This project implements a machine learning model to predict the presence of heart disease in patients based on clinical data.
It serves as a diagnostic tool to help identify high-risk individuals using key health indicators.
Project Overview
The goal is to analyze patient metrics such as age, blood pressure, and cholesterol levels to build a reliable classifier. 
The project includes data preprocessing, exploratory analysis, and a functional prediction interface.

File Structure
 * heart.csv: The primary dataset containing patient health records.
 * heart_disease.ipynb: Jupyter notebook with the data cleaning, analysis, and model training workflow.
 * heart_model.pkl: The trained machine learning model saved for deployment.
 * App_heart.py: A Python script to run the interactive web application for real-time predictions.
 * Requirements.txt: List of Python dependencies required to run the project.

How to Run
 * Clone the repository:
   git clone https://github.com/MohamedElsayedElbadwy/heart_disease_prediction.git

 * Install dependencies:
   pip install -r Requirements.txt

 * Start the app:
   streamlit run App_heart.py

Key Features
 * Detailed Exploratory Data Analysis (EDA).
 * Predictive modeling using classification algorithms.
 * User-friendly dashboard for clinical risk assessment.
